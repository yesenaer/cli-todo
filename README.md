# cli-todo

CLI tool to track to-do lists and items.

## setup
- clone the repo
- (suggested) create a virtual environment
- install the dependencies using `pip install .`
    - or include optional dependencies `pip install .[test]` 

## test
- run `pytest .` from root of repo
